### Design Notes
- Use layered architecture style
- Show dependencies with arrows
- Group related technologies
- Use different colors for each layer

---

## Usage Instructions

1. **For PowerPoint/Google Slides:**
   - Copy the ASCII diagrams
   - Recreate using shapes and arrows
   - Apply color scheme as suggested
   - Add icons where appropriate

2. **For Mermaid Diagrams:**
   - Convert to Mermaid syntax
   - Use in markdown presentations
   - Render with Mermaid.js

3. **For Draw.io/Lucidchart:**
   - Use as reference for structure
   - Recreate with professional shapes
   - Apply consistent styling

4. **For Code-Based Diagrams:**
   - Use Graphviz DOT syntax
   - Use PlantUML
   - Generate programmatically
